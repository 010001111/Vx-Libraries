Assemble Disassemble Engine 86-32

Библиотека поддерживает набор общих инструкций до P6(686), режимы 16/32 бита.
Код библиотеки базонезависимый, без использования внешних функций.

Компилятор: fasm/tasm/masm/nasm


 ФАЙЛЫ:

ADE86.DSP	\ Для сборки проекта в MSVC++6.0
ADE86.DWP	/
MAIN.CPP	Программа оболочка

ADE86.ASM	Ассемблер/дизассеблер
ADE86.H		Заголовочный файл для подключения к C/C++

ADE86.INT	Описание внутренних структур

ADEINC.BIN	Готовый к подключению бинарный файл.

MAIN.ASM	Пример использования в fasm.

./TEST		Набор тестов для ассемблера/дизассемблера

./TEST/Makefile	Для запуска нужна утилита make.exe и fc.exe

ADE86.EXE	Ключи:
		ade86			Запуск интеракивного ассемблера
		ade86 1.asm		Компиляция и листинг на консоль
		ade86 1.asm 1.out	Компилировать в файл 1.out
		ade86 -u 1.out		Дизассемблировать файл 1.out

		 Для дизассемблера/ассемблера допускаются директивы
		use16		Смена разрядности
		use32
		org 100h	Установка текущего смещения
		 Только для дизассмеблера:
		goto 10		Перейти к смещению в файле
